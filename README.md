# How to start on a new mac

## Applications

- Xcode
- VSCode
- iTerm2
- LastPass
- Docker for Mac

## Command line installs

- Install brew
- `brew install shiftit`
- `brew install yarn`
- Install neovim
- Install asdf
  - node
  - ruby
  - python

## Config

- Change shiftit shortcuts
- `~/.config/init.vim`
- `~/.aliases`
